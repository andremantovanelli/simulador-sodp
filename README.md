# Simulador S&OP — Privalia

App Streamlit para simulação de capacitação logística e reforecast de vendas.

## Como rodar localmente (quando tiver Python)
```
pip install -r requirements.txt
streamlit run app.py
```

## Como publicar no Streamlit Cloud
1. Suba os arquivos para um repositório GitHub
2. Acesse share.streamlit.io
3. Conecte o repositório e aponte para app.py

## Arquivos necessários
- `app.py` — código principal
- `requirements.txt` — bibliotecas
